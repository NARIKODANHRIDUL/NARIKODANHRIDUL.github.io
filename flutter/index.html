A chess timer Android app with all basic features like setting timer, pause, reset, changing orientations etc.
NeriQuest Chess Timer is free, easy to use, highly customizable with modern UI.
Set the Timer of your choice with Fischer bonus time option. Three choices available for Fischer Bonus Time are :
  • 10 seconds
  • 5 seconds
  • 0 seconds


Press and hold on either of the Timer button to pause both timer together. A wide variety of color options for changing color of each elements individually.

All Features in one go:
  • modern UI
  • highly customizable Color theme
  • Customization in orientation of timer display
  • User friendly interface
  • Pause timer by long pressing on either timer button
  • Set timer of your choice
  • Fischer Bonus Time
  • Move counter
  • Circular progress indicator to show remaining time
  • New design for active timer
  • Well Explained User Guide
  • Minimum ads
  • Dark theme
