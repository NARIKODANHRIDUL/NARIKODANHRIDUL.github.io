The Hardware badge
  • Design is inspired from IC3 logo
  • Get list of challenges by connecting to putty through laptop
  • Contain all challenges other than PCB Reversing and Soldering
  • It has a :
    • ESP32-S3
    • Type C port
    • set of 3x2 pins
      • at bottom
      • for attaching the Add-On badge
    • EEPROM
    • and some other necessary components
The Add-On badge
  • Inspired from police car
    • with LEDs depicting police Siren Lights
  • Contains remaining challenges, i.e., PCB Reversing and Soldering
  • Power supply comes through the bottom pins of main Hardware board
  • After solving the challenge, have to connect to main board for verifying
  • It has :
    • 7 logical gates IC
    • 12 toggle switches
    • An ATtiny85
    • Some LEds and resistors
