USB Samurai
The primary objective of this project is to obtain access to the victim's air-gapped system remotely over the internet, without the victim's knowledge. To achieve this, we are using an nRF52840 Dongle and a Logitech Unifying dongle. The model of the Logitech Unifying Dongle we used for this was the C-U0007 and C-U0008. The Logitech Unifying receiver is a small dedicated USB wireless receiver, based on the nRF24L-family of RF (Radio Frequency) devices. Logitech Unifying receivers (LURs) are often included in wireless Logitech keyboards, mice, etc.

Attacker gets the powershell access of the victim, which can further used in transfering the data from victims PC to attacker's PC. Through RF we can only tranfer text data, hence when we want to send a file which is not of .txt foromat we can convert the file into a base64 encoded data which can be sent through this air-gapped system and convert back to its orginal file format.
